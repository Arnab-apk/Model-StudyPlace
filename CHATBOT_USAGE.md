# Chatbot Usage Guide

## üéØ How to Use the Chatbot

### Opening the Chatbot
1. Look for the **cyan/purple circular button** in the bottom-right corner of the page
2. Click the button to open the chat window
3. The chatbot will greet you with an introductory message

### Asking Questions
1. Type your question in the **text input field** at the bottom
2. Press **Enter** or click the **Send button** (arrow icon)
3. Wait for the AI to think (you'll see a loading indicator)
4. The response will appear in the chat window

### Mobile Usage
- The chatbot is fully optimized for mobile devices
- The button and window automatically resize for smaller screens
- All touch gestures work naturally
- Keyboard automatically appears when you tap the input field
